# LearnFlow — Аналитический кейс (School of Analytics 2025)

## Situation
EdTech-платформа LearnFlow внедряет новую ML-систему персональных рекомендаций. Существующая модель показывала низкое покрытие каталога (~3.4%) и удерживала пользователей в "информационном пузыре".  

## Task
Оценить новую ML-модель и дать рекомендацию о её внедрении, основываясь на анализе данных, рисков и дизайне A/B-теста.  

## Action
- Провёл анализ данных о пользовательской активности и метаданных курсов (Python, pandas, SQL).  
- Рассчитал метрики эффективности и справедливости модели (coverage, retention, bias).  
- Провёл causal-анализ влияния нерелевантных рекомендаций на вовлечённость.  
- Спроектировал A/B-тест с Guardrail-метриками для безопасного rollout.  

## Result
- Модель увеличивает покрытие каталога с 3.4% до 42.5%.  
- Выявлен bias из-за неполных данных (76% курсов без метки сложности).  
- Для Advanced-пользователей показ нерелевантного контента снижает вовлечённость на 15.1 п.п.  
- Дана рекомендация: **осторожное внедрение (Go) при контроле Guardrail-метрик и доразметке данных**.
